Wee Hold
========
A LudumDare 27 theme of "10 seconds"

When it happened
----------------
August 23rd - 25th 2013 at http://www.ludumdare.com/compo/ludum-dare-27/?action=preview

What it is
----------
Unity3D 4.x implementation of hold your pee so you go for exactly 10 seconds
